# 🔊 FS-AUDIO Sound Library

This directory contains free sound samples for the FS-Sampler instrument plugin.

## 808 Drum Kit

Free TR-808 samples are available from multiple sources:

### Recommended Free Sources:

1. **Samples From Mars - Free 808 Pack**
   - URL: https://samplesfrommars.com/products/free-808-tape-samples
   - License: Free for commercial use
   - Includes: Kick, Snare, Hi-hats, Toms, Clap, Cowbell, Conga, Rim

2. **99Sounds - Free TR-808 Sample Pack**
   - URL: https://99sounds.org/drum-samples/
   - License: Creative Commons
   - High-quality processed 808 sounds

3. **Bedroom Producers Blog - Free 808 Samples**
   - URL: https://bedroomproducersblog.com/tag/free-808-samples/
   - Various free 808 kits available

4. **Freesound.org - User-uploaded TR-808**
   - URL: https://freesound.org/search/?q=tr-808
   - License: Various Creative Commons licenses
   - Community-contributed samples

## How to Add Samples

1. Download samples from the sources above
2. Place them in `public/sounds/drums/808/`
3. Name them descriptively (e.g., `808-kick.wav`, `808-snare.wav`)
4. The sampler will auto-detect and load them
